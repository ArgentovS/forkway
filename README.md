В данном проекте произведена попытка выбрать с помощью нейронной сети заявленные в исследуемых текстах элементы.
Для просмотра исследования и структуры нейтронной сети необходимо открыть файл `model_240323.ipynb`.
В папке `'.Data/'` размещены исходные данные и json-файл с результатом работы нейронной сети.
В папке `'package220323/'` размещены, вспомогательные функции, написанные при подготовке данных для обучения нейронной сети.
